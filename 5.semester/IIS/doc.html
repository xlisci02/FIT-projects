Implementácia

Implementácia zahŕňa prvky použité z knižnice Bootstrap 3.3.7. Riešenie obsahuje niekoľko PHP skriptov, rozumne pomenovaných podľa ich funkcií. Napríklad:

  • index.php - úvodný skript pre prihlásenie do systému
  • logout.php - skript pre odlhásenie užívateľa zo systému
  • add_animal.php - pridanie zvieraťa do systému
  • owner_profile.php - profil majiteľa zvieraťa
  • ...

Instalace

  • Nakopírujte všetky súbory na server
  • Inicialuzujte databázu a naplňte vzorovými dátami (skript db.sql)
  • Potreba MySQL verzie >5.5.62 a PHP minimálne 5.3.29
